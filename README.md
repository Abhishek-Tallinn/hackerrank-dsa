## ΏθΥΕ File Naming Convention

Each solution file is named after the problem in `snake_case`:

```
two_sum.py
reverse_linked_list.java
count_triplets.py
```

Each file includes a header comment with context:

```python
# Problem: Two Sum
# Link: https://www.hackerrank.com/challenges/...
# Difficulty: Easy
# Time Complexity: O(n)
# Space Complexity: O(n)
```

## ΏθγΑ Getting Started

### Run a Python solution
```bash
cd python/arrays
python3 two_sum.py
```

### Run a Java solution
```bash
cd java/arrays
javac TwoSum.java && java TwoSum
```
